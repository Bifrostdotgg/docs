The Heimdall dashboard is a web interface where server administrators configure every feature of the bot. Regular members do not need dashboard access unless they have been granted it explicitly.

## Logging in

Navigate to **[bot.bifrost.gg](https://bot.bifrost.gg)** and click **Sign in with Discord**. Discord will ask you to authorise the dashboard application, which reads your username, avatar, and the list of servers you manage. No message or content access is requested.

Once signed in, you will land on the guild selector.

## Guild selector

The guild selector lists every Discord server where you have the Manage Server permission and Heimdall is present. Click a server's card to open its dashboard.

If you manage a server but Heimdall has not been added to it, the card will appear greyed out with an option to invite the bot.

## Overview page

The overview page is the first thing you see after selecting a server. Rather than a fixed layout, it is a fully customisable widget dashboard. Heimdall provides a library of information widgets — from server member counts and ticket queues to Minecraft server status and modmail volume — and you arrange them to suit how you work.

To customise the layout, click **Edit layout** in the top-right corner of the overview page. In edit mode you can:

- **Add widgets** — click the **+** button to open the widget picker and place widgets from the library
- **Move widgets** — drag any card to a new position on the grid
- **Resize widgets** — drag the resize handle to change a card's dimensions
- **Remove widgets** — hover a card and click the remove button to take it off the grid

The widgets available to you depend on which plugins are enabled on your server and what dashboard permissions your role has been granted. Widgets for plugins that are not active will not appear in the picker.

## Navigation

The left sidebar lists every available plugin. Each entry links to that plugin's configuration and management section. Sections with sub-pages expand when clicked to reveal tabs such as **Configuration**, **Analytics**, **Categories**, and so on.

The sidebar is split into groups matching the sections in this documentation: Moderation, Support, Community, Voice, Games, Integrations, AI, and Utilities.

## Permissions in the dashboard

Your access to dashboard sections depends on what permissions your Discord roles have been granted. If you cannot see a section, you either do not have the relevant permission or the plugin has not been enabled. Administrators with the Discord **Administrator** permission always have full access.

See [Permissions](/permissions) for a full explanation of how the permission system works.

## Making changes

Changes made in the dashboard take effect immediately. There is no publish or save-all step — each section has its own **Save** button that applies changes to that section only.

<Note>
The bot does not need to be restarted when you change configuration. Changes are picked up automatically.
</Note>
