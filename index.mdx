Heimdall is a Discord bot by Bifröst built around a plugin system. Each feature is a plugin that can be enabled or disabled independently. Configuration happens through the web dashboard at [bot.bifrost.gg](https://bot.bifrost.gg), where server administrators control everything from moderation rules to ticket categories to game economy settings.

This documentation covers every feature available to both server administrators and regular members. If you are evaluating Heimdall, see the [pricing page](https://bifrost.gg/heimdall/pricing) for a comparison of available plans.
