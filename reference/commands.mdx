Every slash command available in Bifröst is listed below, grouped by plugin. Commands marked with **Default allow** are available to all members without any permission configuration. All other commands require the matching permission key to be granted to the member's role in the dashboard.

## Moderation

| Command | Description | Default allow |
|---|---|---|
| `/warn <user> <reason>` | Issue a warning infraction | No |
| `/ban <user> <reason>` | Ban a member or user from the server | No |
| `/unban <user>` | Remove a ban | No |
| `/kick <user> <reason>` | Remove a member from the server | No |
| `/mute <user> <duration> <reason>` | Apply a Discord timeout | No |
| `/purge count <count>` | Delete the most recent N messages | No |
| `/purge time <duration>` | Delete messages sent in the last duration | No |
| `/lock [channel]` | Prevent members from sending in a channel | No |
| `/unlock [channel]` | Restore sending permissions to a channel | No |
| `/channelban <user> [channel]` | Ban a member from a specific channel | No |
| `/infractions <user>` | View a member's infraction history | No |
| `/setpoints <user> <points>` | Set a member's active infraction point total | No |
| `/automod` | Manage automod rules | No |
| `/sticky create` | Create a sticky message in a channel | No |
| `/sticky delete [channel]` | Remove a sticky message | No |
| `/appeals config` | Configure the appeals form | No |
